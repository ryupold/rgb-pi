dhtmlxSlider API Methods

dhtmlxSlider Object

  • attachEvent attaches an event handler to slider
  • detachEvent removes an event handler
  • disable disables dhtmlxSlider
  • enableTooltip enables/disables tooplips ( enabled by default )
  • getValue gets value of slider control
  • init initialization of dhtmlxSlider object
  • linkTo links slider to other control
  • setImagePath sets path to images
  • setMax sets new TO value
  • setMin sets new FROM value
  • setOnChangeHandler sets onChange handler
  • setSkin sets the skin for dhtmlxSlider
  • setStep sets new “ST value
  • setValue sets value of slider control
